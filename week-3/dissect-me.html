Hey everyone! So if this is all new to you, don't worry, it's definitely new for me too. I've never used any kind of version control or code storing (when would I need to!?). But I've learned so much in this first week already, and I'm excited to dig into more.

The biggest concepts we delved into in our first course unit is the idea of version control. The key take away of version control is the fact that it's a method to keep you from screwing yourself over by introducing bugs or code that isn't ready, and overwriting all your previous hard work. It's basically a system of gating to make you slow down and not be a complete idiot. Not only that, but if you *were* to introduce a bug, version control allows you to go back to previous versions before you made any mistakes.
